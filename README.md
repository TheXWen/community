# 问答社区项目
### 系统的简单介绍：
##### 使用了SpringBoot + Bootstrap写的一个问答社区项目，主要实现的功能有用户的登录、登录后可以添加和修改提问、回复问题、搜索问题、查看我的问题和收到的回复、查看相关问题。
##### 用户登录使用了gitee的第三方登录功能。添加和修改问题引入了Markdown，使编辑和查看更舒服,实现图片上传功能。回复实现了问题的回复和二级回复。我的回复实现未读提示，我的回复数在通知处显示。
##### 项目部署在阿里云服务器上 ~~[问答社区项目](https://thexwen.cn)~~ 服务器过期已停！
